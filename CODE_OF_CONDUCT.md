# Code of Conduct

## Our Commitment

We are committed to creating a welcoming, respectful, and inclusive environment for everyone, regardless of experience level, background, or identity.

Participation in this project should be a positive experience for all contributors.

---

# Expected Behavior

Contributors are expected to:

- Be respectful and considerate.
- Welcome constructive feedback.
- Collaborate professionally.
- Focus discussions on improving the project.
- Assume good intentions during technical discussions.
- Help create a positive and inclusive community.

---

# Unacceptable Behavior

The following behavior is not acceptable:

- Harassment or discrimination of any kind.
- Personal attacks or insulting language.
- Trolling or intentionally disruptive behavior.
- Publishing another person's private information without permission.
- Any conduct that creates an unwelcoming environment.

---

# Communication Guidelines

When participating in issues, pull requests, discussions, or code reviews:

- Keep conversations professional.
- Provide constructive criticism.
- Explain technical decisions respectfully.
- Focus on ideas rather than individuals.

Healthy technical disagreement is encouraged when conducted respectfully.

---

# Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Maintainers may remove, edit, or reject comments, commits, pull requests, issues, or other contributions that violate these guidelines.

Repeated or severe violations may result in temporary or permanent restrictions from participating in the project.

---

# Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it privately to the project maintainers.

Reports will be reviewed fairly and handled with appropriate confidentiality whenever possible.

---

# Scope

This Code of Conduct applies to all project spaces, including:

- GitHub Issues
- Pull Requests
- Discussions
- Code Reviews
- Repository Documentation
- Any other official project communication channels

---

# Acknowledgements

This Code of Conduct is inspired by established open source community standards and has been adapted for the needs of this project.
