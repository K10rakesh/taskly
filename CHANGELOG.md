## Versioning

This project follows Semantic Versioning.

### Major Release

Breaking changes.

Example:

```
1.0.0 → 2.0.0
```

---

### Minor Release

New features that remain backward compatible.

Example:

```
1.0.0 → 1.1.0
```

---

### Patch Release

Bug fixes and small improvements.

Example:

```
1.0.0 → 1.0.1
```
